Ce dossier ne sert à rien de fonctionnel. Il est juste plus pratique pour stocker les fichiers .contrapp. Libre à vous de l'utiliser

Lorsque vous lancez l'application, celle-ci va lire les données contenues dans un fichier .contrapp dans le même dossier que là où est le .exe
S'il n'y en a pas, l'application génère un fichier .contrapp vide

Les fichiers .contrapp peuvent être déplacés, transférés à quelqu'un d'autre, ou renommées (tant que ça se fini par .contrapp)

Cela permet de transférer les données de son application à quelqu'un d'autre sans problème. Ou bien d'avoir plusieurs versions de l'application
